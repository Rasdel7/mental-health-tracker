# Mental Health & Mood Tracker 🧠

Track daily mood, energy, sleep and stress
with personalized insights and trend analysis.

## Live Demo
[Click here](YOUR_STREAMLIT_URL)

## Features
- Daily mood, energy, sleep, stress logging
- Mood gauge with color coded levels
- 7-day trend visualization
- Correlation heatmap between metrics
- Calendar heatmap view
- Journal with mood-tagged entries
- Auto-generated personalized insights
- Activity impact analysis
- Streak tracking

## Privacy
All data stored locally on your device.
Nothing sent to any server.

## Tools Used
- Python, Streamlit, Plotly, Pandas

## How to Run Locally
pip install streamlit pandas numpy plotly
streamlit run app.py
